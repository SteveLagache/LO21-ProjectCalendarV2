Detailed Description

Projet.

Un projet est caractérisé par : des tâches un titre (permettant d’en saisir la nature) une date de disponibilité (date à laquelle on pourra commencer le projet) une date d'écheance (date à laquelle le projet doit être terminé) L’échéance d’un projet est une borne supérieure de l’ensemble des échéances des tâches de ce projet.
