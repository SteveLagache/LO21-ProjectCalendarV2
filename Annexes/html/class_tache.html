Detailed Description

Tache.

Une tâche ne peut être associée qu’à un seul projet. Une tâche (unitaire ou composite) est caractérisée par : un titre (permettant d’en saisir la nature) une date de disponibilité (date à laquelle on pourra commencer la tâche) une date d'écheance (date à laquelle la tâche doit être terminée.
